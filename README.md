решето Эратосфена
Решето Эратосфена — это классический алгоритм нахождения всех простых чисел до заданного целого числа \(N\), приписываемый древнегреческому математику Эратосфену Киренскому.Как работает алгоритмЗапись: Выписывается ряд чисел от \(2\) до \(N\).Поиск: Находится первое невычеркнутое число (изначально это \(2\)) — оно объявляется простым.Вычеркивание: Из ряда вычеркиваются все числа, кратные найденному (например, \(4, 6, 8 \dots\)).Повтор: Процедура повторяется для следующего невычеркнутого числа, пока не будут проверены все числа вплоть до \(\sqrt{N}\).Все оставшиеся невычеркнутыми числа — простые.

Оценка эффективностиВремя работы: \(O(N \log \log N)\). Это один из самых быстрых и эффективных методов для поиска простых чисел.Память: \(O(N)\).
